Paragraph element in HTML is used to display block of text.
It is written using the <p> tag.

Key Characteristics of <p>
    - Used for normal text content
    - Always starts on a new line
    - Adds margin (space) by default
    - Block-level element

When to use <p>
    - Writing explanations
    - Adding descriptions
    - Displaying article or blog content
    - Showing readable text on a webpage.
