JavaScript ES6 Data Types
    - In JavaScript (ES6 and above), there are 8 main data types.

    They are divided into:
        - Primitive Types (Simple values)
        - Reference Type (object)

1. Primitive Data Types (7 Types)
    Primitive = stores actual value directly.

    1. Number
    Represents:
        - Integers
        - Decimals
        - Special values (Infinity, NaN)

        let age = 25;
        let price = 99.99;
        let result = 10 / 0;    - Infinity

    2. String
    Text values.

        let name = "Jade";
        let message = "Hello ${name};
    
    3. Boolean
    True or False.

    let isOnline = true;
    let isLoggedIn = false;

    Used in:
        - Conditions
        - Comparisons
        - Logic Operations

    4. Undefined
    A variable declared but not assigned.

        let x;
        console.log(x);     - undefined
    
    5. Null
    Represents intentional empty value.

        let selectedUser = null;

        Diffrence:
        - undefined -> not assigned
        - null -> intentionally empty
    
    6. Symbol (ES6 Feature)
    Introduced in ES6, creates unique identifiers

        const id1 = Symbol("id");
        const id2 = Symbol("id");

        Even if description is name, they are unique.

        Used for:

        - Object property keys
        - Advance programming

    7. Bight (ES11 but modern JS)
    Used for very large integers.

        const bigNumber = 123456789012345678901234567890n;

2. Reference Data Type

    8. Object
    Stores collections of data.

    const person {
        name: "Jade",
        age: 25
    }

    Arrays and functions are also objects:

    const numbers = [1, 2, 3];

    function greet() {
        return "Hello";
    }

Primitive vs Reference (Important!)

Primitive:
    - Stored directly
    - Copied by value

    let a = 10;
    let b = a;
    b = 20;

    console.log(a); // 10

Reference
    - Stored by reference (memory address)
    - Copied by reference

    let obj1 = {name: "Jade"}
    let obj2 = obj1;

    obj2.name = "Mark";

    console.log(obj1.name);     // "Mark"
