String concatenation - means joining two or more strings together or form one single string.

Think of it like connecting words in a sentence.

1. Using + Operator (Old Way)

    let firstName = "Jade Mark";
    let lastName = "Gimpao";

    let fullName = firstName + " " + lastName;

    console.log(fullName);

    Output: 
    Jade Mark Gimpao

    Here:
    - + joins the strings
    - " " adds a space between them

2. Using Template Literals (ES6 Way - Recommended)

    let price = 100;
    let message = `The total price is ${price} dollars.`;

    Much cleaner.
